## 核心專案

### NIS AI客服平台
* 實作技術：Python,LangGraph,RAG
* 說明：結合大型語言模型與檢索增強生成技術，開發針對護理資訊系統的自動化客服平台。

### Android學習清單
* 實作技術：Java, SQLite
* 說明：運用Java語言與SQLite資料庫開發之行動端應用程式，落實物件導向設計與資料持久化。

### 資安威脅監測
* 實作技術：Suricata
* 說明：負責網路安全環境部署，並透過撰寫自定義偵測規則進行威脅獵捕。
